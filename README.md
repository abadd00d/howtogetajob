### Сколько будет собеседований и с кем?

Обычно процесс состоит из **3 этапов**:

1.  **(до 30 мин) HR-скрининг**.
    Созвон с HR-менеджером. Проясняем первые вопросы по вакансии и опыту. Если все ок — переходим к технике.
2.  **(1 час) Техническое собеседование**.
    Присутствуют: HR-менеджер, нанимающий менеджер (начальник отдела) и иногда Lead-инженер.
    _В редких случаях технических этапов может быть два, если не успели обсудить всё._
3.  **(до 30 мин) Оффлайн встреча в офисе**.
    Финальный этап. Смотрим офис, задаем последние вопросы и окончательно понимаем, подходим ли мы друг другу.

### Что нужно для собеседования?

Чтобы технические заминки не влияли на твою оценку, пожалуйста, проверь это заранее:

1.  **Камера**.
    Сделай так, чтобы твое лицо было хорошо видно: сядь напротив окна или включи свет. Расположи камеру на уровне глаз — это выигрышный ракурс.
2.  **Звук**.
    Лучше использовать гарнитуру, чтобы звук из динамиков не создавал эхо. Убедись, что тебе не будут мешать посторонние шумы.
3.  **Софт**.
    Заранее установи [Yandex.Telemost](https://telemost.yandex.ru/) (приложение удобнее веб-версии).
    > **Важно:** Выдай приложению все права до начала звонка: видео, звук, **шаринг экрана** (на собеседовании нужно будет демонстрировать рабочий стол).
4.  **Терминал**.
    Тебе понадобится терминал Linux для демонстрации опыта работы с этой операционной системой. Можно использовать VM или хостовую операционку на железе, главное, чтобы можно было выполнять команды.

### Тестовое задание и рекомендации

**Тестовое задание:** Выдается для позиций уровня Middle/Senior. Обычно занимает до нескольких часов. Если есть вопросы по условию — смело пиши.

**Рекомендации:** Для позиций уровня Middle/Senior мы запрашиваем рекомендации с прошлых мест работы.
