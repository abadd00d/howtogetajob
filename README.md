# Нулевая толерантность обману

Один из принципов нашей компании – честность. С клиентами, партнерами и с коллегами. Мы кране не приемлем попытки кандидатов каком-то образом обмануть работодателя.

Мы доверяем кандидатам, но в тоже время мы в курсе современных трендов при поиске работы и не нанимаем кандидатов, которые обманывают работодателя. Считаем нужным предупредить об этом заранее, чтобы мы не тратили наше общее время.

Вот некоторые из способов, которые используют недобросовестные кандидаты:
1. Накрутка опыта или несуществующий опыт. Сюда попадает как накрутка опыта по времени, так и указание несуществующих компаний, так и указание реальных компаний, в которых ты не работал.
2. Скрытие своих реальных данных. Например, указать не реальные имя/фамилию или другие данные.
3. Использовать различные инструменты для прохождения собеседонвания: AI, онлайн конференс связь с помощником и т.п.

Первые два пункта будут проверены при заполенении анкеты на СБ + трудовая (будьте готовы предоставить выгрузку подписанной трудовой с Госуслуг). 3й пункт не принесет результата, т.к. один из этапов проводиться оффлайн.
